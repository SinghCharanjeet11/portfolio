# Developer Portfolio

Minimal, modern portfolio built with Next.js App Router, Tailwind CSS, and Framer Motion.

## Setup

```bash
npm install
npm run dev
```

## Customize

Edit the following files:
- `components/Hero.js` - Name, role, description
- `components/About.js` - Professional summary
- `components/Projects.js` - Project details
- `components/Skills.js` - Skills list
- `components/Contact.js` - Email and social links

## Deploy

```bash
npm run build
```

Deploy to Vercel, Netlify, or any static hosting platform.
